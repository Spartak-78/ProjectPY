# Book Tracker

**Автор:** Тельпов Юрий Вячеславович

## Описание программы

Book Tracker — простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Поддерживает фильтрацию по жанру и количеству страниц. Все данные сохраняются в файл `books.json`.

## Инструкция по запуску

1. Установите Python 3.x.
2. Установите библиотеку Tkinter (обычно встроена в Python).
3. Скачайте или склонируйте репозиторий.
4. Запустите файл `book_tracker.py`:
   ```bash
   python book_tracker.py
   ```

## Примеры использования

- Добавьте книгу через форму и нажмите «Добавить книгу».
- Используйте фильтры для поиска книг по жанру или количеству страниц.
- Данные сохраняются автоматически при добавлении новой книги.
